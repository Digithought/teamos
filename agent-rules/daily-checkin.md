# Daily Check-in

Your daily check-in has fired. This is a lightweight cycle to ensure you stay current even when you have no explicit tasks, messages, or events queued.  

## What to Do

This is your chance to be proactive rather than reactive.

## Guidelines

- **Keep it brief.** This should be a quick scan, not a deep dive. If you discover substantial work, create a todo and handle it in a future cycle.
- **Don't duplicate the self-assessment.** This is not reflective evaluation — it's proactive awareness.
- **Update state.md** only if something meaningful changed.
- **It's okay to do nothing.** If everything looks good and there's no work to generate, just go back to sleep to save resources. Not every check-in needs to produce output.
